# ByteForge

**Advanced Macro tool with voice control & intelligence.**

Record macros, spam keys, auto-click, run voice commands, schedule tasks, and automate anything on Windows.

## Features
- Voice wake word ("ByteForge") + natural commands
- Powerful macro recorder & player
- Auto clicker + key spammer with hotkeys
- Smart Assistant that understands what you want
- Anti-idle mouse nudge
- Scheduled macros
- Speech-to-Text anywhere
- Beautiful neon UI + tray icon
- Auto-update support
- User accounts & settings

## Quick Start
1. Download the latest `.exe` from [Releases](https://github.com/azaan-ship-it/ByteForge/releases/latest)
2. Run it (Windows may show a warning — this is normal for indie software)
3. Create an account or log in
4. Press **F9** to toggle the window

**Default Hotkeys:**
- Toggle Macro: `Shift + Enter`
- Toggle Clicker: `Ctrl` (press 3 times)
- Panic Stop: `Alt`
- Show/Hide Window: `F9`
